# 环境安装与运行

这份说明只覆盖当前仓库已经公开的内容，也就是表格型强化学习笔记和对应实验。

## 推荐方式

```bash
conda env create -f environment.yml
conda activate ReinforcementLearning
```

当前环境只包含运行现有实验需要的最小依赖：

- `python`
- `numpy`
- `matplotlib`
- `gymnasium`
- `tqdm`

## 备用方式

如果你已经有自己的 Python 环境，也可以直接安装根目录依赖：

```bash
pip install -r requirements.txt
```

## 运行当前实验

运行 `FrozenLake` 上的 `Q-Learning`：

```bash
cd experiments/01-frozenlake-tabular-q
python train.py --episodes 4000 --render-final-policy
```

查看 Q 值沿路径传播的教学脚本：

```bash
cd experiments/01-frozenlake-tabular-q
python trace_q_updates.py --episodes 6
```

运行 `CliffWalking` 上的 `SARSA`：

```bash
cd experiments/02-cliffwalking-tabular-sarsa
python train.py --episodes 800 --render-final-policy
```

运行 `SARSA` 与 `Q-Learning` 的对比脚本：

```bash
cd experiments/02-cliffwalking-tabular-sarsa
python compare_sarsa_q_learning.py --episodes 800
```

运行 `Blackjack` 上的 `Monte Carlo`：

```bash
cd experiments/03-blackjack-monte-carlo
python train.py --episodes 200000 --render-final-policy
```

## 运行环境说明

- 当前公开实验都是表格方法，CPU 即可运行
- 当前仓库不需要 `PyTorch` 或 CUDA
- 如果后续加入 `DQN`、`PPO` 等内容，再单独补充深度学习依赖

## 常见问题

### `gymnasium` 导入失败

先确认当前终端使用的是正确环境：

```bash
conda activate ReinforcementLearning
```

### 图像没有显示或保存失败

确认 `matplotlib` 已安装，并检查当前目录是否有写权限。
